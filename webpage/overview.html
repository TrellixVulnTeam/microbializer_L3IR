Introduction

The technological advances in the last decade brought with them opportunities for large-scale mining and analysis of bacterial species data. These analyses posses many challenges including data-mining steps, such as gene annotation and orthologs detection, sequence alignment, accurate phylogenetic tree reconstruction and genome wide association studies. These steps as well as additional analysis-specific computations require the execution of multiple bioinformatics tools, making this process cumbersome, tedious and prone to errors due to manual handling.

What is M1CR0B1AL1Z3R?

The M1CR0BIALIZ3R webserver is a "one-stop shop" for conducting such microbial genomics data analyses via a simple graphical user interface (GUI) without any installations, making this processes efficient and easy and allows scientists to analyze hundreds of bacterial genomes, with a click of a button. An example of features which are implemented in M1CR0BIALIZ3R include:
  • Extracting all open reading frames (ORFs) [1];
  • Detecting and extracting orthologous sets sequences from ORFs [2, 3];
  • Analyzing presence-absence patterns of genes;
  • Align orthologous sets sequences [4];
  • Inferring the core-genome of the species in the dataset;
  • Reconstructing a maximum-likelihood phylogenetic tree based on the inferred core-genome [5];
  • Providing plots of different statistics of the data;

    • Input

    The input for the webserver is a zipped folder (either .zip or .tar.gz) in which each file is a fasta nucleic acid (.fna) file containing genomic sequences of a different species (whether fully sequenced or as contigs)

    Output and methodology

    The output of the webserver contains the following:
  • A reliable species phylogeny tree (both in a Newick format and as a ready-to-publish figure);
  • A curated table of orthologous sets;
  • A fasta file for each orthologous set containing the UNALIGNED sequences of the set members;
  • A fasta file for each orthologous set containing the ALIGNED sequences of the set members;


    • All these analyses are presented graphically, as can be demonstrated in our gallery section.
